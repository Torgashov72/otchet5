Добавление полей login и password_hash в базу данных

Подключился к MySQL через SSH: mysql -u u82361 -p Выбрал базу данных: USE u82361; Добавил поля командой: ALTER TABLE application ADD COLUMN login VARCHAR(50) UNIQUE, ADD COLUMN password_hash VARCHAR(32); Проверил структуру таблицы: DESCRIBE application;

Первая отправка формы и генерация логина/пароля

При первой отправке формы происходит: Валидация всех полей формы Генерация уникального логина (функция generate_login()) Генерация случайного пароля (функция generate_password()) Хеширование пароля через md5() Сохранение всех данных в БД Отображение логина и пароля пользователю один раз
